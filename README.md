# Data Science

Data Science repository for the EnergyTrading2026 project seminar. Contains forecasting and optimization work of the project seminar.

## Teams

- **Forecasting**
- **Optimization**

## Setup

```bash
git clone git@github.com:EnergyTrading2026/datascience.git
cd datascience
pip install -e .
```

## Project Structure

```
src/            Production code (forecasting + optimization)
notebooks/      Jupyter notebooks for exploration and prototyping
tests/          Automated tests
docs/           Problem definitions, methodology notes
data/           Data (raw and cleaned)
```

## Workflow

1. Create a feature branch from `main`
2. Make your changes
3. Open a pull request
4. Get 1 approval
5. Merge
